## 📂 Featured Projects

### 📊 [Superstore Sales Analytics — Power BI + PostgreSQL](https://github.com/HrushhikeshMate/Superstore-End-to-End-Data-Analytics-Project)
> **End-to-end analytics pipeline**: Python ingestion → PostgreSQL star schema → Power BI dashboard

- Designed a **star schema** (fact + dimension tables) in PostgreSQL for the Sample Superstore dataset
- Wrote **CTEs and window functions** to compute YoY sales trends, discount impact on margins, and regional profitability gaps
- Built a **multi-page Power BI dashboard** with drill-through filters, KPI cards, and dynamic slicers
- **Key finding**: High-discount orders in the Furniture segment drove a negative margin in 3 of 4 regions

**Stack:** `Python (Pandas)` · `PostgreSQL` · `Power BI` · `DAX`

---

### 🏀 [NBA Game Outcome Prediction — LightGBM Classifier](https://github.com/HrushhikeshMate/Basketball-Game-Outcome-Prediction-)
> **MSc Thesis project**: Predicting NBA game winners using team performance statistics

- Engineered features from historical team stats (pace, offensive/defensive rating, recent form)
- Trained and tuned a **LightGBM classifier**, achieving ~**65% accuracy** and **~0.700 AUC**
- Evaluated model against baseline (home-team always wins) — consistent outperformance across seasons
- Documented methodology, feature importance, and model limitations in a structured report

**Stack:** `Python` · `Scikit-learn` · `LightGBM` · `Pandas` · `Matplotlib`
